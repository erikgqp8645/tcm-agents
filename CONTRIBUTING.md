# 贡献指南

感谢你对伤寒明理论解构与溯源专家项目的关注！

## 如何贡献

### 1. 改进 Prompt

核心 Prompt 位于 `prompts/system-prompt.md`。如果你发现：
- 某些病证的拆解不够深入
- 溯源规则可以更精确
- 防幻觉机制可以更强

欢迎提交 PR。

### 2. 添加解析示例

在 `examples/` 目录下添加更多《伤寒明理论》篇章的解析示例：
- 复制 `examples/悸-analysis.md` 作为模板
- 用同样的格式解析其他篇章（如"喘"、"渴"、"头痛"等）
- 确保溯源结果经过人工校验

### 3. 完善参考文献

在 `references/` 目录下：
- 添加更完整的《伤寒论》条文索引
- 补充《金匮要略》条文对照
- 整理历代注家对同一问题的不同解读

### 4. 报告问题

如果你发现：
- Prompt 导致幻觉（编造不存在的条文号）
- 某些病证的拆解明显错误
- 溯源结果不准确

请提交 Issue，附上：
- 输入原文
- 输出结果
- 你认为的正确结果

## 提交规范

### Commit Message

```
feat: 添加"渴"篇解析示例
fix: 修正桂枝汤溯源为第12条而非第13条
docs: 更新README中的使用说明
prompt: 优化防幻觉规则的措辞
```

### PR 标题

- `feat: 新功能/示例`
- `fix: 修复错误`
- `docs: 文档更新`
- `prompt: Prompt优化`

## 注意事项

1. **溯源准确性**：所有新增的溯源结果必须经过人工校验，引用可靠的《伤寒论》/《金匮要略》原文版本
2. **不要修改原始古籍文本**：输入原文应保持原貌，不做任何现代标点或删改
3. **中英文并重**：核心文档保持中英双语
