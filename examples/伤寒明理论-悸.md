# 伤寒明理论·悸 — 深度拆解与溯源

## [原文段落]

> 悸。伤寒悸者．何以明之．悸者心忪是也．筑筑踢踢然动．怔怔忪忪．不能自安者是矣．心悸之由．不越二种．一者气虚也．二者停饮也．伤寒二三日．心中悸而烦者．小建中汤主之．少阴病四逆．其人或悸者．四逆散加桂五分．是气虚而悸者也．饮水多．必心下悸．是停饮而悸者也．其气虚者．由阳气内弱．心下空虚．正气内动而为悸也．其停饮者．由水停心下．心为火而恶水．水既内停．心不自安．则为悸也．又有汗下之后．正气内虚．邪气交击．而令悸者．与气虚而悸者．则又甚焉．太阳病发汗过多．其人叉手自冒心．心下悸．太阳病若下之．身重心下悸者．不可发汗．少阳病不可吐下．吐下则悸而惊．少阳病不可发汗．发汗则谵语．此属胃．胃和则愈．胃不和则烦而悸．是数者．皆汗后挟邪者．与其气虚而悸者．有以异也．或镇固．或化散之．皆须定其气浮也．又饮水过多．水饮不为宣布．留心下．甚者则悸．金匮要略曰．食少饮多．水停心下．甚者则悸．饮之为悸．甚于他邪．虽有余邪．必先治悸．何者．以水停心下．若水气散．则无所不之．浸于肺则为喘为咳．传于胃则为哕为KT ．溢于皮肤．则为肿．渍于肠间．则为利下．不可缓之也．经曰．厥而心下悸．宜先治水．与茯苓甘草汤．后治其厥．不尔水渍于胃．必作利也．厥为邪之深者．犹先治水．况其邪气浅者乎．医者可不深究之．

## [深度病理拆解矩阵]

### 🔴 对比组一：气虚之悸 vs 停饮之悸

| 维度 | **气虚之悸** | **停饮之悸** |
|---|---|---|
| **病位** | 心下（胃脘部），本质在**中焦脾胃**，气不生血则心失所养 | 心下（胃脘部），本质在**中焦水停**，水邪犯心 |
| **病性** | **虚证**——阳气内弱，正气不足 | **实证为主**——水饮停聚为患 |
| **气机演变** | 中阳不足 → 脾胃虚寒 → 心下空虚 → 正气无所依附而内动 → 悸 | 水饮入胃 → 脾失运化 → 水停心下 → 心属火恶水 → 水火相搏 → 悸 |
| **方证举例** | ①伤寒二三日心中悸而烦 → **小建中汤**（建中补虚，调和气血）；②少阴病四逆或悸 → **四逆散加桂**（透达郁阳，温通心脉） | 饮水多必心下悸 → **茯苓甘草汤**类（温阳化饮，利水宁心） |
| **🔑 鉴别金指标** | ①**悸而烦**——烦是气血不足、心神失养的特征性伴随；②**四逆**——阳郁不达四末；③手不冒心（自觉心空） | ①**饮水多史**——有明确的水饮过量诱因；②**心下有水声**（虽本文未明言，但临床可察）；③甚者则悸，轻者不悸 |

### 🔴 对比组二：单纯气虚之悸 vs 汗下后挟邪之悸

| 维度 | **单纯气虚之悸** | **汗下后挟邪之悸** |
|---|---|---|
| **病机** | 阳气内弱，正虚不甚 | 正气内虚 **＋ 邪气交击**——虚实夹杂，病势更重 |
| **严重程度** | 较轻，虚悸而已 | **则又甚焉**——正虚邪扰，悸动更剧 |
| **原文判据** | 心中悸而烦 | 叉手自冒心 / 身重心下悸 / 悸而惊 / 烦而悸 |
| **治法** | 补虚建中即可 | **或镇固、或化散**——需根据邪气性质（水？表邪？少阳郁热？）灵活处置，但前提是**定其气浮** |

### 🔴 对比组三（成氏暗线）：水饮为悸之"轻者" vs "甚者"

- **轻者**：饮水多，水停心下，悸而不甚 → 尚可缓图
- **甚者**：水气弥漫，浸肺为喘咳、传胃为哕噫、溢肤为肿、渍肠为利下 → **不可缓之**
- **🔑 核心原则**：饮之为悸，**甚于他邪**。虽有余邪（如厥、如外感），**必先治水**。此为"急则治标"之极致——因为水邪的扩散性远超其他邪气。

## [仲景经文溯源 (Source Tracing)]

### 📜 溯源点1：伤寒二三日，心中悸而烦者，小建中汤主之

- **出处**：《伤寒论》第102条（辨太阳病脉证并治中）
- **仲景原文**："伤寒二三日，心中悸而烦者，小建中汤主之。"
- **印证解说**：成无己原文几乎**逐字引用**此条。这是成氏"气虚之悸"立论的核心支柱。"悸而烦"是关键鉴别点——**悸是心失所养，烦是气血两虚**，二者同见，指向中焦虚寒、气血化源不足。小建中汤以饴糖为君，温中补虚，建立中气，使气血生化有源，则悸烦自除。

### 📜 溯源点2：少阴病四逆，其人或悸者，四逆散加桂

- **出处**：《伤寒论》第318条（辨少阴病脉证并治）
- **仲景原文**："少阴病，四逆，其人或咳，或悸，或小便不利，或腹中痛，或泄利下重者，四逆散主之。"
- **印证解说**：原文言四逆散，**未言加桂**。成无己此处引为"加桂五分"，是成氏根据临床经验在原方基础上的加减法——因为**悸**在本条是"或然证"，由少阴阳郁、气机不畅所致，加桂枝意在**温通心阳、助气外达**。这体现了成氏对仲景方的灵活运用，而非死守原方。

### 📜 溯源点3：太阳病发汗过多，叉手自冒心，心下悸

- **出处**：《伤寒论》第64条（辨太阳病脉证并治中）
- **仲景原文**："发汗过多，其人叉手自冒心，心下悸，欲得按者，桂枝甘草汤主之。"
- **印证解说**：成无己准确引用此条，作为"汗后挟邪而悸"的典型案例。**叉手自冒心**——病人双手交叉按压心胸部位，是心阳骤虚、心无所主的本能反应，比"悸而烦"更重一筹。原文中"欲得按"三字尤其关键，揭示此悸属**虚悸**，需按压以求自安。成氏将此条归入"汗后挟邪"类，强调其比单纯气虚之悸"更甚"。

### 📜 溯源点4：太阳病若下之，身重心下悸者，不可发汗

- **出处**：《伤寒论》第49条
- **仲景原文**："脉浮数者，法当汗出而愈。若下之，身重，心下悸者，不可发汗，当自汗出乃解。所以然者，尺中脉微，此里虚，须表里实，津液自和，便自汗出愈。"
- **印证解说**：成氏援引此条说明"误下后"损伤里气，正虚不运，水湿内停而身重、心悸。关键禁令是"**不可发汗**"——因为正气已虚，再汗则更伤其阳。这与成氏"汗下后挟邪者，与气虚而悸者有以异"的分类完全一致。

### 📜 溯源点5：少阳病不可吐下 / 不可发汗

- **出处**：《伤寒论》第264条、第265条（辨少阳病脉证并治）
- **仲景原文（264条）**："少阳中风，两耳无所闻，目赤，胸中满而烦者，不可吐下，吐下则悸而惊。"
- **仲景原文（265条）**："伤寒，脉弦细，头痛发热者，属少阳。少阳不可发汗，发汗则谵语，此属胃，胃和则愈，胃不和则烦而悸。"
- **印证解说**：成无己将此两条合论，作为"汗后挟邪之悸"的少阳专题。264条强调少阳病误吐下 → **悸而惊**（悸为心虚，惊为神怯）；265条强调误发汗 → 谵语 → 胃不和则**烦而悸**。此处"烦而悸"与小建中汤证的"悸而烦"用字顺序不同，暗含病机差异：小建中汤是虚在先（悸）而后生烦；少阳误汗是胃热在先（烦）而后及心（悸）。

### 📜 溯源点6：饮水多，必心下悸

- **出处**：《伤寒论·辨脉法》或《金匮要略·痰饮咳嗽病脉证并治》
- **原文大意**："食少饮多，水停心下，甚者则悸，微者短气。"
- **印证解说**：成无己此处将《伤寒论》与《金匮要略》的条文合引，作为"停饮之悸"的核心依据。"食少饮多"揭示了病因——脾胃运化能力弱（食少）而水液摄入过多（饮多），水无以化，停于心下。

### 📜 溯源点7（最重磅）：厥而心下悸，宜先治水，与茯苓甘草汤

- **出处**：《伤寒论》第356条（辨厥阴病脉证并治）；部分内容亦见于《金匮要略·呕吐哕下利病脉证治》
- **仲景原文**："伤寒厥而心下悸，宜先治水，当服茯苓甘草汤，却治其厥。不尔，水渍入胃，必作利也。"
- **印证解说**：这是成无己全文压轴的关键条文。它确立了中医临证中**"急则治标"**的极致原则：**即使是厥（四肢厥冷，属重证），只要同时有心下悸（水停之兆），也必须先治水。** 因为水邪具有弥漫性——"浸于肺为喘咳，传胃为哕噫，溢肤为肿，渍肠为利下"——不先截断水势，后续变证无穷。成无己以"厥为邪之深者，犹先治水，况其邪气浅者乎"作为反问收尾，极具说服力。

## [医家核心奥义总结]

成无己本篇的核心贡献在于：**将"悸"从一个模糊的症状，提炼为一个具有明确鉴别诊断框架的证候。**

1. **二分法立论**：悸不越"气虚"与"停饮"二者，简洁有力。
2. **加一层叠加**：汗下后挟邪之悸，是正虚与邪扰的叠加态，比单纯气虚之悸更重，治法也更复杂（镇固/化散并用）。
3. **治水优先原则**：在"停饮"与"他邪"并存时，**先治水**——这是仲景"急则治标"思想的典范。
4. **文法精妙**："厥为邪之深者，犹先治水，况其邪气浅者乎"——以退为进的论证手法，极具临床警示力。
